# Process Tracing Handoff

## What this project is

`process_tracing` is a local, typed process-tracing pipeline for historical causal analysis. It extracts events, hypotheses, evidence, diagnostic tests, absence findings, Bayesian support, synthesis, and refinement from source text and source-packet context, then renders the result as an auditable report and a browser workbench. The goal is PhD / think-tank / academic quality, not generic qualitative coding.
